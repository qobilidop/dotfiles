# dotfiles
My dotfiles bootstrapped by [Dotbot](https://github.com/anishathalye/dotbot)

## Usage

On my laptop:
```bash
git clone --recursive https://github.com/qobilidop/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.py base.laptop
```

On a remote server:
```bash
git clone --recursive https://github.com/qobilidop/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.py base.server
```

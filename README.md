# dotfiles

My personal dotfiles mostly for development on macOS

## Usage

```bash
$ mkdir -p .local/opt
$ git clone git@github.com:qobilidop/dotfiles.git ~/.local/opt/dotfiles
$ cd ~/.local/opt/dotfiles
$ make
```

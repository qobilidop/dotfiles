# dotfiles

My dotfiles are bootstrapped by [Dotbot](https://github.com/anishathalye/dotbot).

## Usage

On my Mac:
```bash
git clone --recursive https://github.com/qobilidop/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.py base.local.macos
```

On a remote server:
```bash
git clone --recursive https://github.com/qobilidop/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.py base.remote
```
